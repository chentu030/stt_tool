# Cadence Product Roadmap

語音驅動的知識工作區。功能借鏡 Notion、Heptabase、Word、PowerPoint，依 **基礎 → 進階** 分階段導入。

## 產品對照摘要

| 來源 | Cadence 要學的 |
|------|----------------|
| **Notion** | 區塊編輯、資料庫、範本、協作、AI |
| **Heptabase** | 卡片庫、白板空間思考、PDF 標註、Journal、雙向連結 |
| **Word** | 文件樣式、找取代、修訂、長文結構、匯出 |
| **PowerPoint** | 簡報結構、母片思維、一鍵產投影片、簡報模式 |

---

## Notion（基礎 → 進階）

### 基礎
- 頁面／子頁面巢狀
- 區塊編輯（文字、標題、清單、待辦、引用、分隔線）
- `/` 指令插入區塊
- 拖曳排序、縮排
- 圖片／檔案上傳
- 全文搜尋
- 深淺色、側欄導覽
- 匯出 Markdown／PDF
- 分享連結（唯讀／可編輯）

### 中階
- Toggle、Callout、Code、表格
- Embed（YouTube、Figma…）
- `@` 提及頁面／人
- 資料庫：表格／看板／日曆／圖庫／時間軸
- 屬性（文字、選項、日期、人員、檔案）
- 篩選、排序、分組
- 範本（Template）
- 留言／評論
- 版本歷史
- 表單（Form → 寫入資料庫）
- 頁面驗證／封存

### 進階
- 資料庫 Relation／Rollup／Formula
- 子任務、依賴、自動化
- Charts／Dashboard 檢視
- Tabs 區塊、簡報模式
- Wiki 結構、權限分級
- Notion AI（改寫、摘要、Autofill、Meeting Notes）
- Agent／跨 App 搜尋
- 公開網站發布、API／整合

---

## Heptabase（基礎 → 進階）

### 基礎
- 卡片（筆記）庫
- 區塊式編輯器
- 白板：放卡片、移動、縮放
- 雙向連結 `[[ ]]`
- 全域搜尋
- 日常 Journal（每日卡片）
- 離線／跨裝置同步

### 中階
- 白板箭頭連線、區塊分組（Section）
- 子白板
- Mind map
- PDF 卡片＋標註／Highlight → 高亮卡片
- 圖片／影音／音訊卡片
- Web Clipper
- 白板內搜尋、標籤整理
- 行動端快速捕捉（含語音）

### 進階
- Readwise／Zotero 匯入
- 多來源同板研究
- AI Chat（卡片／白板／PDF 為 context）
- AI Tutor／學習流程
- CLI／外部 AI 工具串接
- 大規模知識庫秒搜

## Word（基礎 → 進階）

### 基礎
- 標題樣式、內文、粗斜底線
- 項目符號／編號清單
- 段落對齊、行距、縮排
- 尋找／取代
- 貼上選項（保留／合併／純文字）
- 拼字檢查
- 圖片、超連結
- 匯出 PDF

### 中階
- 表格
- 頁首／頁尾、頁碼、目錄
- 樣式集／主題
- 追蹤修訂、註解、`@` 提及
- 即時共編
- 翻譯、輔助使用

### 進階
- 郵件合併、長文件大綱
- 比較文件、版本還原
- 範本／樣式鎖定
- 引用與書目
- Copilot／Agent 起草與整篇重構

---

## PowerPoint（基礎 → 進階）

### 基礎
- 投影片新增／排序、版面配置
- 文字方塊、項目符號
- 圖片、形狀、主題色
- 投影片播放
- 匯出 PDF／圖片

### 中階
- 母片（Slide Master）
- 動畫／轉場（克制）
- 圖表、對齊、群組
- 備註、演講者檢視
- 嵌入影音、共編
- 章節（Sections）

### 進階
- 品牌套件／範本合規
- 動態圖表、互動超連結
- 錄製旁白
- Copilot：從大綱／文件一鍵產簡報

## Cadence 導入順序（實作優先）

### Phase A｜文件基礎（進行中）
1. **區塊編輯器**（標題／清單／待辦／引用）
2. **`/` 指令**插入區塊
3. **拖曳排序**區塊
4. 樣式列（粗斜、對齊、找取代加強）
5. 圖片／附件、超連結
6. 匯出 MD／PDF／DOCX

### Phase B｜知識組織
7. `[[wikilink]]`、反向連結
8. 標籤、資料夾／收藏
9. 全庫搜尋＋篩選
10. Journal（每日筆記）
11. 範本（會議／課堂／訪談）

### Phase C｜結構化
12. 簡單資料庫（表格檢視）
13. 屬性：狀態、日期、來源 job
14. 看板／日曆檢視
15. 留言與版本歷史

### Phase D｜空間思考
16. 白板：卡片、連線、分組
17. PDF 標註 → 高亮卡片
18. 子白板、Mind map

### Phase E｜簡報輸出
19. 筆記／逐字稿「一鍵產簡報大綱」
20. 簡報模式
21. 匯出 PPTX／PDF 投影片

### Phase F｜智能與協作
22. 共編、權限、分享
23. AI：摘要／改寫／依白板或 PDF 問答
24. Relation／自動化、Dashboard
25. Agent、對外整合
